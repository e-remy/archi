Une fonction applicative représente un groupe cohérent de comportement interne d'un composant applicatif.

Une fonction applicative décrit le comportement interne d'un composant applicatif. Le comportement interne de la fonction est invisible pour l'utilisateur d'un composant applicatif.

Le comportement d'une fonction applicative est exposé à l'extérieur au travers d'un ou plusieurs services. Une fonction applicative peut réaliser des services applicatifs.

Les services applicatifs d'autres fonction applicatives et services d'infrastructure peuvent être utilisés par une fonction applicative.

Une fonction applicative peut accéder aux objets de données applicatifs.

Un composant applicatif peut être affecté à une fonction applicative.

Le nom d'une fonction applicative devrait de préférence être un substantif évoquant une action, par exemple "comptabilité".

Catégorie :

Applicatif. Comportement.

Exemples :

Comptabilité, Facturation, Création de contrat, Calcul de prime, Administration financière.

Relations types :

Utilisé par - service applicatif
Déclenchement - fonction applicative
Accès - objet de données applicatif
Affectation - composant applicatif
