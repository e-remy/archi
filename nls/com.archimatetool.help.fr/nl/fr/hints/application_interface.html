Une interface applicative déclare comment un composant se connecte à son environnement.

Une interface applicative spécifie comment la fonctionnalité d'un composant peut être accédée par d'autres composants. Elle peut aussi spécifier quelle fonctionnalité le composant requiert de son environnement (NDLR : ce second cas est quasi inutilisé).

Une interface applicative spécifie comment la fonctionnalité d'un composant peut être accédée par d'autres composants (interface fournie), ou bien quelle fonctionnalité le composant requiert de son environnement (interface requise).

Une interface applicative expose un service applicatif à son environnement l'environnement. Un même service applicatif peut être exposé au travers de différentes interfaces.

Une interface applicative peut être affectée à des services applicatifs ou à des services métier.

Le nom d'une interface applicative est de préférence un substantif.

Catégorie :

Applicatif. Structure active.

Exemples :

Echange de données de transaction.

Relations types :

Affectation - service applicatif, service métier
