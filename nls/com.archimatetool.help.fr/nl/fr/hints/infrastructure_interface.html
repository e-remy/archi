Une interface d'infrastructure est le point d'accès où la fonctionnalité offerte par un nœud peut être accédée par d'autres nœuds ou composants applicatifs.

Une interface d'infrastructure spécifie comment les services d'infrastructure d'un nœud peuvent être accédés par d'autres nœuds (interface fournie), ou quelle fonctionnalité le nœud requiert de son environnement (inteface requise).

Une interface d'infrastructure expose un service d'infrastructure à son environnement.

Une interface d'infrastructure peut faire partie d'un nœud par une relation de composition.

Un service d'infrastructure peut être affecté à une interface d'infrastructure, qui expose le service à l'environnement.

Le nom d'une interface d'infrastructure devrait de préférence être un substantif.

Catégorie :

Technique. Structure active.

Exemples :

Logiciel client.

Relations types :

Affectation - service d'infrastructure
Composition - nœud
