Un lieu est un point ou une étendue conceptuels dans l'espace.

Le concept de lieu est utilisé pour modéliser la distribution d'éléments structurels tels que les acteurs métier, les composants applicatifs et les machines. Cette distribution est modélisée grâce à une relation d'affectation du lieu à l'élément de structure. Indirectement, un lieu peut aussi être affecté à un élément de comportement, afin d'indiquer où le comportement est exécuté.

Catégorie :

Métier. Structure active.

Exemples :

Bureaux principaux, agence locale, pièce.

Relations types :

Affectation - rôle ou acteur métier
Agrégation - acteur métier
