Une interface métier déclare comment un rôle métier se connecte à son environnement.

La fonctionnalité apportée par un rôle métier est exposée à son environnement par le moyen d'une ou plusieurs interfaces métier. A l'inverse, un rôle métier peut être consommateur de la fonctionnalité fournie par d'autres rôles métier via une interface métier.

Une interface métier spécifie comment la fonctionnalité d'un rôle métier peut être utilisée par d'autres rôles métier (interface fournie), ou quelle fonctionnalité les rôles métier requièrent de leur environnement (interface requise).

Une interface métier expose un service métier fourni par un rôle métier ou une collaboration métier à son environnement. Un service métier peut aussi être exposé au travers de différentes interfaces.

Le nom d'une inteface métier devrait de préférence être un substantif.

Catégorie :

Métier. Structure active.

Exemples :

Télépone, e-maill, centre d'appels, formulaire Web, assistance support.

Relations types :

Affectation - acteur, rôle, collaboration ou service métier
