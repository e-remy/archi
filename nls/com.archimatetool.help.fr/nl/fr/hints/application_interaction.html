Une interaction applicative est une unité de comportement exécutée par la collaboration de plusieurs composants.

Une interaction applicative décrit le comportement extérieurement visible qui est appliqué par des composants qui participent à une collaboration applicative.

Une interaction applicative peut également spécifier le comportement extérieurement visible nécessaire pour réaliser un service applicatif.

Une collaboration applicative peut être affectée à une interaction applicative.

Une interaction applicative peut réaliser un service applicatif.

Les services applicatifs et les services d'infrastructure peuvent être utilisés par une interaction applicative.

Une interaction applicative peut accéder à des données applicatives.

Le nom d'une interaction applicative devrait de préférence être un verbe.

Catégorie :

Applicatif. Comportement.

Exemples :

Administrer les transactions.

Relations types :

Affectation - collaboration applicative
Réalisation - service applicatif
Utilisation par - service applicatif, service d'infrastructure
Accès - objet de données applicatif
