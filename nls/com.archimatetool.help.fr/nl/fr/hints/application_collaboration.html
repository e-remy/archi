Une collaboration applicative est un configuration (temporaire) de plusieurs composants qui coopèrent pour exécuter ensemble des interactions applicatives.

Une collaboration applicative spécifie quels composants coopèrent pour réaliser une tâche.

Le comportement collaboratif, incluant par exemple le pattern de communication entre ces composants, est modélisé par une interaction applicative.

Le nom d'une collaboration applicative devrait de préférence un substantif.

Catégorie :

Applicatif. Structure active.

Exemples :

Deux composants applicatifs collaborent dans le cadre d'une transaction administrative: un composant de Comptabilité et un de Facturation. Cette collaboration exécute l'interaction applicative "Administrer les transactions".

Relations types :

Affectation - interaction applicative, interaction métier
Utilisé par - interface applicative
Composition - interface applicative
