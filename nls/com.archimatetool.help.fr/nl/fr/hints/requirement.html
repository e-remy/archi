Une exigence est défini comme la formalisation d'un besoin qui doit être réalisé par un système.

Les exigences modélisent les propriétés des éléments qui sont nécessaires pour atteindre la "finalité" modélisée par les objectifs. Dans cette perspective, les exigences représentent les "moyens" d'atteindre les objectifs.

Catégorie :

Motivation

Exemples :

"Affecter un assistant personnel", "Fournir le service de portefeuille en ligne", "Fournir un service d'information en ligne"

Relations types:

Composition, agrégation, spécialisation, influence
