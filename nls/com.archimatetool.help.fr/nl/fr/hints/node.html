Un nœud est une ressource de traitement sur laquelle des artefacts peuvent être déployés pour exécution.

Les nœuds exécutent et traitent les artefacts, qui sont les représentations des composants et des objets de données applicatifs.

Les nœuds sont utilisés pour modéliser les serveurs d'application, les serveurs de données ou les stations de travail.

Les nœuds peuvent être constitués de sous-nœuds représentant les machines physiques et les environnements d'exécution pour les artefacts.

Les nœuds peuvent être connectés entre eux par des chemins de communication. Les artefacts peuvent être affectés, i.e. déployés, sur des nœuds.

Un nœud peut être constitué de plusieurs sous-nœuds. Les artefacts déployés sur un nœud peuvent être dessinés directement à l'intérieur du nœud ou bien connectés via une relation d'affectation.

Le nom d'un nœud devrait de préférence être un substantif.

Catégorie :

Technique. Structure active.

Exemples :

Mainframe, ferme de serveurs Unix, serveur d'application, firewall.

Relations types :

Affectation - artefact, réseau
