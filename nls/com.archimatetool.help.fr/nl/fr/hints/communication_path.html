Un chemin de communication est un lien entre plusieurs nœuds, au travers duquel ces nœuds peuvent échanger des informations.

Un chemin de communication est utilisé pour modéliser la communication logique entre nœuds.

Il est réalisé par un ou plusieurs réseaux, qui représentent les liens physiques de communication.

Un chemin de communication connecte plusieurs nœuds. Un chemin de communication est réalisé par un ou plusieurs réseaux.
