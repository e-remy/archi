Un objet métier est une unité d'information utile et compréhensible dans une perspective métier.

Un objet métier est utilisé pour modéliser un type d'objet dont plusieurs instances peuvent exister au sein de l'organisation. Dans ce cas, il peut être réalisé en tant qu'objet de données applicatif ou en tant que représentation. Il peut aussi être spécialisé par un autre objet métier.

Les objets métier sont passifs. Ils ne déclenchent ni n'exécutent aucun processus.

Catégorie :

Métier. Structure passive.

Exemples :

Facture, Fiche client, Dossier étudiant, Dossier de suivi, Registre.

Relations types :

Accès - processus fonction, interaction, événement ou service métier
Réalisation - représentation, objet de données applicatif
