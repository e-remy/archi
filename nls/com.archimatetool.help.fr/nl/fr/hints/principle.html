Un principe est une propriété normative de tous les systèmes dans un contexte donné, ou bien la façon dont ils sont réalisés.

Les principes définissent les propriétés attendues d'un système. Un principe définit une propriété générale qui s'applique à n'importe quel système dans un certain contexte. Un principe est motivé par un objectif.

Catégorie :

Motivation

Exemples :

"Les systèmes doivent être orientés client", "L'expérience client doit être bonne", "Les collègues doivent être informés"

Relations types :

Composition, agrégation, spécialisation, influence
