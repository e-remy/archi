Un moteur est défini comme quelque chose qui crée, motive et alimente le changement dans une organisation.

Les moteurs peuvent être internes, auquel cas ils sont habituellement associés à une partie prenante. Ils peuvent également être externes.

Le nom d'un moteur devrait de préférence être un substantif.

Categorie :

Motivation

Exemples :

"Satisfaction client", "Conformité réglementaire", "Profitabilité", "Changements économiques", "Changements législatifs"

Relations types :

Composition, agrégation, spécialisation, influence
