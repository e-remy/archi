Une machine est une ressource de traitement informatique sur laquelle des artefacts peuvent être déployés pour exécution.

Une machine est une spécialisation d'un nœud qui représente une ressource physique avec des capacités de traitement. Elle est typiquement utilisée pour modéliser des systèmes matériels tels que des serveurs mainframe, des PCs ou des routeurs.

Elles sont habituellement regroupées dans un même nœud avec des logiciels système. Les machines peuvent être composites, i.e. être constituées de plusieurs sous-machines.

Les machines peuvent être connectées entre elles par des réseaux. Des artefacts peuvent être affectés aux (i.e. déployés sur les) machines. Une machine peut être constituée de sous-machines.

Un logiciel système peut être affecté à une machine. Un nœud peut contenir une ou plusieurs machines.

Le nom d'une machine devrait de préférence être un substantif faisant référence au type de matériel.

Catégorie :

Technique. Structure active.

Exemples :

Serveur de fichiers, routeur, mainframe, PC fixe, IBM Système Z.

Relations types:

Composition - machine, nœud
Affectation - logiciel système, artefact
